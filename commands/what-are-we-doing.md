# What Are We Doing?

Give the user a quick, high-level read on this session. **This is query-only — do not create, edit, or delete any files, and do not run commands that change state.**

Based on the conversation so far, respond with:

1. **A 2-3 sentence summary** of what you believe the user is trying to accomplish in this session overall.
2. **If the session is long** (many turns or several distinct tasks), add **one more sentence** summarizing the specific task the user is currently on.

Guidelines:
- Keep it brief and plainly worded — no preamble, no headings, just the sentences.
- Describe the goal and intent, not a blow-by-blow log of every tool call.
- If the session has just started or intent is unclear, say so in one sentence instead of guessing.
